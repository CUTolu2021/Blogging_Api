## Blogging API

A RESTful API for managing blog posts, categories, and tags.
https://roadmap.sh/projects/blogging-platform-api

## Description

This project is a RESTful API for managing blog posts, categories, and tags. It provides a simple and intuitive way to create, read, update, and delete blog posts, as well as manage categories and tags.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API Endpoints

POST /posts: Create a new blog post
PUT /posts/:id: Update an existing blog post
DELETE /posts/:id: Delete an existing blog post
GET /posts/:id: Get a single blog post
GET /posts: Get all blog posts
GET /posts?term=tech: Filter blog posts by a search term

## Database Schema

The database schema consists of three tables: posts, categories, and tags. The posts table has the following columns:

id (primary key)
title
content
category_id (foreign key referencing the categories table)
tag_id (foreign key referencing the tags table)
The categories table has the following columns:

id (primary key)
name
The tags table has the following columns:

id (primary key)
name
